== Description ==

Show a floating box on every page that make visitors required to choose a qookie level. After choosing and authorizing a level the information is logged in the database.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click on the menu item thats was added and turn it on
1. Change the settings on the settings page
